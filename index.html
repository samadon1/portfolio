Projects

I build impactful and efficient software systems to deliver social good. Here are a few of my most recent projects.

SignTalk - Sign Language Translation

SignTalk is sign language translation system that allows deaf and dumb patients to communicate effectively with their doctors by using the power of computer vision and deep learning. It is capable of both translation of speech-to-sign-language and sign-language-to-speech.

Healthcare acccessibility Map App

The healthcare accessibility map app computes the approximate travel time from every point to the nearest hospital (darker is longer) in Ghana.

Nonprofit Finder

This project connects donors and grantmakers to non-profit organizations. A platform to list verified NGOs/non-profit organizations in Ghana. The ultimate goals of this project is to verify NGOs operating in Ghana, expose verified NGOs to donors and volunteers looking, give donors/individuals conducting due diligence on a non-profit organization a way to do so seamlessly on the platform

English-Twi Translator

Twi translator is a machine translation app that generates translations of never-before-seen real-world input sentences from English to Twi. I implemented the model from scratch by training a sequence-to-sequence transformer in Python

Heart MRI Image Segmentation

The aim of this project is to build a predictive model that could classify each pixel in a cardiac MRI image based on whether the pixel is a part of the left ventricle (LV) or not. In medical imagery analysis it is often important to separate the pixels corresponding to different types of tissue, blood or abnormal cells so that we can isolate a particular organ. In this project I used the TensorFlow machine learning framework to train and evaluate an image segmentation network using a medical imagery dataset.

Land-Cover-Classification

This project introduces an example of a supervised land cover classification with Sentinel-2 data. The aim is to classify a Sentinel-2 image into four different land cover classes: water surfaces, artificial surfaces, low vegetation and tree cover.

Local Library Website

This platform manages the catalog for a local library. The website provides an online catalog for a small local library, where users can browse available books, reserve books and manage their accounts

Ultrasonic Digital distance meter

Digital distance meter is a non-contact distance measurement app thats uses an ultrasonic sensor to measure the distance of target objects. The sensor data is sent over a realtime database which is then accessed and displayed by the app
